# Data-Cleaning-Project

## Overview
This project focuses on cleaning and preprocessing a dataset to make it ready for analysis. The goal was to handle missing values, correct data types, and ensure consistency across all columns.

## Dataset
The dataset contains information with multiple columns including numerical and categorical data.
Initial data had missing values and inconsistent formats.

## Steps Performed
- Data Exploration
- Handling Missing Values
- Data Type Conversion
- Data Validation

## Final Result
- Clean dataset with 0 missing values
- Correct and consistent data types
- Ready for analysis or visualization

## Tools & Libraries
- Python
- Pandas
- NumPy

## Key Learnings
- Importance of handling missing data
- Data type correction for accurate analysis
- Building a clean data pipeline
